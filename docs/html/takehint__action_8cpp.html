Detailed Description

Node that manages the robot taking the hints.

Author
Maria Luisa Aiachini

Publishes to:
°/hint_list

Subscribes to:
°/visualization_marker °/oracle_hint

Description:

This node manages the robot taking the hint: at first it manages the first round, where the robot tries both the possible positions of the hint to save them: after the first round the robot's arm goes directly to the right height. When the node receives the hint from the /oracle_hint topic there are some checks to verify that the hint received is totally valid and then it fills the vectors containing the complete hints. It finally uses the /hint_list publisher to send the complete list of hypotheses.
