Detailed Description

Node for managing the hints: checking if there are complete and consistent hypoteheses anche checking the winning one.

Clients:
°/oracle_solution

Subscribes to:

Description:

This node is used to check the hints. It takes the hypotheses list, through /hint_list topic, it checks if there is any complete and consistent hypotheses, if so, it checks if it is the winning one, after printing it in natural language.
